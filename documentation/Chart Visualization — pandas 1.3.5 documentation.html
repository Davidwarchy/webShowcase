Creating and Manipulating Sets

This page demonstrates various operations on set objects. The programming language used is python.

We create a set object:

In [1]: s1 = {1,2,3} 

In [2]: s2 = {3,4,5}

Note

A set can be created from other objects such as lists and numpy arrays

Union

A union operation on two sets combines both sets, picking out only the unique members in those sets

In [3]: s3 = s1.union(s2)

Unions from Other Objects

We can also create set objects from various types of objects

Dictionary

In [1]: d1 = {'a':1, 'b':2, 'c':3}

In [2]: d2 = {'c':3, 'd':4, 'e':5}

In [3]: s1 = set(d1)

In [4]: s1 = set(d2)

In [5]: s3 = s1.union(s2)

List

In [1]: l1 = ['a','b','c','d']

In [2]: l2 = ['b','a','c','h']

In [3]: l1.extend(l2)

In [4]: s = set(l1)

Array

In [1]: import  numpy as  np

In [2]: n1 = np.array([1,2,3])

In [3]: n2 = np.array([1,2,3])

In [4]: n3 = np.concatenate([n1,n2])	

In [5]: s = set(n3)
